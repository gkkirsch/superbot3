Features

Everything an AI system needs

Spaces

Each space is its own Claude Code instance. Isolated plugins, skills, agents, and settings. Connected to your code repos.

Memory & Knowledge

Karpathy-style knowledge wiki + persistent memory system. Nightly consolidation. Your AI gets smarter over time.

Dashboard

Chat with any space. Manage plugins, files, schedules, settings. Real-time updates via WebSocket.

Self-Improving

Spaces can edit their own skills, create agents, manage schedules. Built-in reflection and learning patterns.

Getting Started

Three commands to running AI

1

Install with one command

curl -fsSL https://raw.githubusercontent.com/gkkirsch/superbot3/main/install.sh | bash

2

Create a space

superbot3 space create "My Project" --code-dir ~/dev/myapp

3

Start everything

superbot3 start — dashboard at localhost:3100
